## Truoc khi build

- Cap nhat version trong firmware.
- Dong bo README va changelog.
- Kiem tra flag debug da tat.

## Artefacts can luu

- File binary cuoi cung.
- Commit hash.
- Cau hinh board da dung de build.
- Log test smoke test.

## Rollback

Neu ban phat hanh firmware cho thiet bi dang hoat dong, luon giu mot ban binary on dinh truoc do de rollback nhanh.
